## 建议用法

1. 先遮住中文解释，只根据日语词条回忆核心含义。
2. 将固定搭配作为整体记忆，不孤立背单词。
3. 对比日常例句和职场例句，观察搭配是否变化。
4. 每次从一页中选择三个表达，用自己的场景重新造句。
