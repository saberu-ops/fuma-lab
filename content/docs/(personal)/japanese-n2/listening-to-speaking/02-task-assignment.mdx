## 学习目标

- 自然盘点尚未完成的任务。
- 根据既定分工向他人委派工作。
- 在接手任务时确认范围、顺序和期限。

## 原文呈现

> **女：** うん。
>
> **男：** あと残っているのは関係者に発表会の案内メールを送ることとポスターの作成かな？案内のメールは僕から送ることになってるからポスターを頼める？
>
> **女：** どの曲をやるかポスターに入れるよね？曲の順番はこの間決めた通りでいい？
>
> **男：** うん、案を作ってくれたらみんなで確認をするから今週中にお願い。
>
> **女：** 了解。

## 逐句解析与翻译

1. **「うん。」**

   - **翻译**：嗯。
   - **作用**：承接上文的对话。

2. **「あと残っているのは関係者に発表会の案内メールを送ることとポスターの作成かな？」**

   - **翻译**：剩下还没做的，应该就是给相关人员发发表会的通知邮件，还有制作海报了吧？
   - **作用**：清晰盘点项目剩余进度，语尾用「かな」表达思考和确认。

3. **「案内のメールは僕から送ることになってるからポスターを頼める？」**

   - **翻译**：通知邮件按计划是由我来发，所以海报能拜托你吗？
   - **作用**：依据既定分工「ことになってる」，自然地委派剩余任务。

4. **「どの曲をやるかポスターに入れるよね？曲の順番はこの間決めた通りでいい？」**

   - **翻译**：海报上要写上表演哪些曲目对吧？曲目的顺序按照前几天定好的那样可以吗？
   - **作用**：接手任务后确认具体细节，避免因信息不一致造成返工。

5. **「うん、案を作ってくれたらみんなで確認をするから今週中にお願い。」**

   - **翻译**：嗯，你把草案做出来之后大家一起确认，所以麻烦在本周内完成。
   - **作用**：明确后续工作流和交付期限。

6. **「了解。」**

   - **翻译**：明白。
   - **作用**：简洁地接受任务与时间节点。

## 核心语块拆解

**核心语块：** `この間決めた通り`（按照前几天决定好的那样）

- `この間（このあいだ）`：前几天、上次，指最近发生过的一段时间。
- `決めた`：动词「決める」的过去式，在这里作定语。
- `通り（とおり）`：表示按照某个既定内容或标准执行。

## TPO 替换与进阶

| 场景 | 表达 |
| --- | --- |
| Casual（熟人、平辈） | この間決めた通りでいい？ |
| Business（上级、正式职场） | 先日決めた通りでよろしいでしょうか。 |

## 实际场景平移

**代码审查场景（Casual）**

> 了解。テストの項目は、この間決めた通りでいい？

**项目需求确认（Business）**

> 承知いたしました。仕様の変更については、先日ミーティングで決めた通りでよろしいでしょうか。

## 输出练习

使用 `〜ことになっている` 和 `〜通り`，完成一段任务确认：

1. 说明自己负责的事项。
2. 请求对方承担另一项任务。
3. 确认执行方式是否沿用之前的决定。
4. 明确草案或结果的提交时间。
