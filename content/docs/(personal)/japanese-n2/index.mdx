## 内容分组

| 分组 | 组织维度 | 放入这里的内容 |
| --- | --- | --- |
| 历年真题 | 考试年月 → 科目 | 原题、音频、字幕、答案、解析、来源与校订状态 |
| 专项学习 | 能力专题 → 课次 | 从多套真题提炼的词汇、语法、听力方法和输出训练 |
| 项目规划 | 建设阶段 | 收录范围、完成标准、工具建设和学习功能 |

同一份材料只保留一个主版本：原始真题归入“历年真题”，跨试卷整理后的知识才进入“专项学习”。专项页面通过链接引用真题，不复制整份原文。
