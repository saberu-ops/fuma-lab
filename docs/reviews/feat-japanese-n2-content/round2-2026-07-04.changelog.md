# Round 2 Changelog: JLPT N2 Core Vocabulary and Grammar

## Scope

documentation area and published the result through the running Fumadocs site.

## Content and Navigation

- Added `core-vocabulary-grammar` as a Japanese N2 topic with its own metadata
  and overview page.
- Split 32 vocabulary entries into three review groups:
  - operations and states;
  - descriptions and change;
  - context and behavior.
- Split 13 grammar entries into three review groups:
  - connections and states;

- Updated the README and Japanese-content authoring guide to include the new
