## Intent

- Use the existing Next.js and Fumadocs application instead of introducing a
  separate publishing system.
- Organize learning content as domain, topic, and lesson so pages remain small,
  addressable, searchable, and easy to extend.
- Keep navigation maintenance low by combining automatic `meta.json` discovery
  with numeric lesson slugs.
- Give each lesson a predictable learning flow from input and analysis to
  comparison, scenario transfer, and active output.
- Keep normal builds and search local and offline.
- Make Chinese explanations, Japanese expressions, and English technical terms
  searchable in the same documentation tree.

## Operational Decisions

- The retained root Markdown is an archive; migrated MDX pages are the
  maintained web-content source.
- Existing topics must contain `meta.json` before the scaffold command writes a
  page, preventing accidental creation of unstructured navigation folders.
- Stable ASCII slugs are separated from translated display titles so title
  changes do not break URLs.
- CJK bigrams are used instead of per-character indexing to support continuous
  text while reducing noisy multi-character search results.
- No new runtime dependency is required for scaffolding or tokenization.

## Explicitly Not Done

- The original root Markdown file was not edited, moved, or deleted.
- No database, CMS, browser editor, account, or server-side learning-progress
  storage was added.
- No audio files, copyrighted examination media, or external content were
  imported.
- No authentication, TLS, reverse proxy, or public deployment was added.
- No dependency version or package-lock entry was changed.
- No routed i18n or automatic translation was added.

## Known Limitations

- CJK tokenization is deterministic n-gram indexing, not morphological analysis.
  It favors practical phrase lookup over linguistic ranking.
- A single CJK character embedded inside a longer uninterrupted run is not
  independently searchable; multi-character words and phrases are the intended
  query unit.
- Topic index cards and lesson counts are maintained manually even though
  sidebar page discovery is automatic.
- The archived root Markdown is intentionally not synchronized with later MDX
  edits.
- `npm ci` reported five moderate dependency advisories from the existing
  locked dependency graph. No forced or breaking audit fix was applied.
