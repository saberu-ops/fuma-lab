# Round 1 Changelog: JLPT N2 Exam Archive Information Architecture

## Scope

Reorganize the Japanese N2 area so it can manage multiple years of exam
materials without mixing source exams with derived study courses. Add a
site-visible roadmap for expanding the current listening exercise into a
maintainable exam archive.

## Changes

- Added `past-exams` as the canonical archive, grouped by exam month and then
  subject.
- Moved the 2024-07 listening page to
  `/docs/japanese-n2/past-exams/2024-07/listening`.
- Added archive and exam-session indexes with explicit completeness states.
- Added `/docs/japanese-n2/roadmap` with phased outcomes, priorities,
  completion criteria, and scope boundaries.
- Separated archive navigation from listening, vocabulary, and grammar study
  topics.
- Added a permanent redirect from the former listening-course URL.
- Updated the authoring guide, personal landing page, and README to document
  the new model.
