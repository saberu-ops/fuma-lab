# Round 4 Changelog: Interactive Listening Page Balance

## Scope

Remove document-oriented chrome from the listening exercise and clarify the
responsibility of each player column.

## Changes

- Added an `interactive` page frontmatter field to the docs schema.
- Interactive pages retain their title and metadata description but omit the
  visible description, Markdown copy button, and view-options control.
- Marked the July 2024 listening exercise as interactive.
- Changed the player to equal-width desktop columns.
- Removed the large current-cue panel from the playback column because the
  transcript column already follows, highlights, and scrolls to the active
  sentence.
- Kept playback, seeking, rate, sentence repeat, transcript visibility, and
  sentence navigation behavior unchanged.
