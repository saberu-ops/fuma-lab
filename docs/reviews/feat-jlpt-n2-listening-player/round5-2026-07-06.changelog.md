# Round 5 Changelog: Viewport-Filling Listening Workspace

## Scope

Use the available desktop viewport without reintroducing redundant page copy,
and turn the added space into practical listening controls.

## Changes

- Made the desktop player fill the viewport below the page title with a safe
  minimum height for shorter screens.
- Stretched both equal-width columns to the same player height.
- Allowed the transcript viewport to grow with the player while preserving its
  independent scrolling.
- Distributed playback controls vertically within the available left-column
  space.
- Added global previous/next question navigation and a current question count
  across all 29 clips.
- Added a compact keyboard shortcut reminder on wide screens.
- Kept mobile layout content-sized with a bounded transcript viewport.
