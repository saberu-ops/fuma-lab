# Round 2 Changelog: SRT Source Integration

## Scope

This round resumes the interrupted follow-up to the JLPT N2 listening player.
It replaces the earlier timing-document conversion path with the supplied
`2024年07月N2听力音频.srt` file as the direct subtitle and clip-boundary source.

## Changes

- `scripts/build-listening-exam.mjs` now parses and validates standard SRT
  blocks directly.
- `lib/data/jlpt-n2-2024-07.json` was regenerated from all 342 source SRT cues.
  The 29 exercise clips contain 264 cues; cues outside configured question
  boundaries remain excluded.
- `public/audio/jlpt-n2/2024-07/*.mp3` was regenerated from the SRT-derived
  boundaries.
- The generated clips span 2,293.806 seconds and occupy 22,957,945 bytes.

The original `202407.md`, `202407.txt`, `202407.mp3`, and the supplied SRT file
remain unchanged as authoring inputs.
